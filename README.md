# SHACL Semantic Roles Demo

Live demo of event-centric knowledge graph with automatic SHACL inference.

## Features

- VerbNet/FrameNet/WordNet integration
- Automatic opaque IRI generation
- D3.js force-directed graph visualization
- Real-time SPARQL inference

## Live Demo

Visit: https://falcontologist.github.io/shacl-demo

## API

Backend API: https://shacl-api-docker.onrender.com

## Tech Stack

- Frontend: Vanilla JavaScript, D3.js
- Backend: Java, TopBraid SHACL API
- Ontology: VerbNet, FrameNet, WordNet, Croftian Constructions
